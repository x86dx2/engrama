# Security Policy

Para reportar uma vulnerabilidade, abra uma issue privada ou contate o mantenedor do repositório por um canal privado acordado no projeto.

Escopo honesto:

- O gate local é cooperativo e serve como atrito útil no fluxo de trabalho.
- O enforcement vinculante depende do check de CI estar ativo como `required`.
- Este repositório não deve receber segredos em commits, fixtures, docs ou arquivos de exemplo.
